# Raycast Scripts 模块

## 模块职责
存放 Raycast 启动器的自定义脚本扩展，用于快速触发系统操作或工具命令。

## 脚本清单
- `claude-code-voice-toggle.sh`: 切换 Claude Code 的语音功能 (Voice TTS)。
- `claude-code-ring-toggle.sh`: 切换提示音 (Ring)。
- `stop-voice-tts.sh`: 立即停止正在播放的 TTS 语音。

## 使用方式
将本目录添加到 Raycast 的 Script Commands 路径中即可使用。
